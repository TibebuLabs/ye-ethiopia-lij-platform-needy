# Ye Ethiopia Lij Platform

A web-based platform for integrated support and sponsorship tracking of orphaned and street children in Ethiopia.

## Features
- Role-based access (Admin, Organization, Sponsor, School)
- Child profile management with duplication prevention
- Sponsorship tracking and transparency
- Intervention logging and reporting
- Real-time notifications
- Multi-language support (planned)

## Tech Stack
- Frontend: React.js, Redux Toolkit, Material-UI
- Backend: Django REST Framework (not included)
- Database: MongoDB
- State Management: Redux Toolkit
- HTTP Client: Axios
- Form Handling: Formik + Yup

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run development server: `npm start`
5. Build for production: `npm run build`
